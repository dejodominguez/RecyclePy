VII. Localization

You can localize the Editor to be displayed in specific language by adding a language include file before the Editor javascript include:

<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">

<script language="Javascript" src="scripts/language/de-DE/editor_lang.js"></script>
<script language="Javascript" src="scripts/innovaeditor.js"></script>
</head>
<body>
<form method="post" action="post.asp" id="Form1">

<textarea id="txtContent" name="txtContent" rows=4 cols=30></textarea>
<script>
var oEdit1 = new InnovaEditor("oEdit1");
oEdit1.REPLACE("txtContent");
</script>
<input type="submit" value="submit">
</form>
</body>
</html>

The current available language include files are:

  • Danish

    <script language="Javascript" src="scripts/language/da-DK/editor_lang.js"></script>

  • Dutch

    <script language="Javascript" src="scripts/language/nl-NL/editor_lang.js"></script>

  • Finnish

    <script language="Javascript" src="scripts/language/fi-FI/editor_lang.js"></script>

  • French

    <script language="Javascript" src="scripts/language/fr-FR/editor_lang.js"></script>

  • German

    <script language="Javascript" src="scripts/language/de-DE/editor_lang.js"></script>

  • Norwegian

    <script language="Javascript" src="scripts/language/nn-NO/editor_lang.js"></script>

  • Chinese (Simplified)

    <script language="Javascript" src="scripts/language/zh-CHS/editor_lang.js"></script>

  • Chinese (Traditional)

    <script language="Javascript" src="scripts/language/zh-CHT/editor_lang.js"></script>

  • Spanish

    <script language="Javascript" src="scripts/language/es-ES/editor_lang.js"></script>

  • Swedish

    <script language="Javascript" src="scripts/language/sv-SE/editor_lang.js"></script>

  • Italian

    <script language="Javascript" src="scripts/language/it-IT/editor_lang.js"></script>

If no language file specified, English version will be displayed.
